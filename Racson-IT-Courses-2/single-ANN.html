What is Artificial Neural Network (ANN) ?

ANN is a subset of machine learning The term ANN is derived from biological neural network that develop the structure of a human brain

All the value today of deep learning is through supervised learnign or learning from labeled data and algorithms. Deep Learning is implemented with the help of deep network of neurons with multiple hidden layers. In this blog you will learn more about Deep Learning.

Biological Neural Network VS Artificial Neural Networks

Artificial Neural Network is a type of neural network which is based on a Feed-Forward strategy. It is called this because they pass information through the nodes continuously till it reaches the output node. This is also known as the simplest type of neural network.

Biological Neural Network (BNN) is a structure that consists of Synapse, dendrites, cell body, and axon. In this neural network, the processing is carried out by neurons. Dendrites receive signals from other neurons, Soma sums all the incoming signals and axon transmits the signals to other cells.

Advantages of ANN.

  • Ability to learn irrespetive of the data (Linear or Non-Linear).

  • ANN is highly volatile and serves best in financial time series forcasting.

Advantages of Biological Neural Network.

  1. The synapses are the input processing element.

  2. It is able to process highly complex parallel inputs.

What is Activation Function in Neural Network ?

Activation function is a mathematical function which adds non-linearity to the neighbours. An activation function decides wheather a neuron's shoul be activated or not. This means that it will decides weather the neuron's input to the network is important or not in the process of prediction using mathematical operations. In the process of building a neural network, one of the choices you get to make is what activation function to use i the hidden layer as well as at the output layer of the network.

Why Activation function is used in Neural Network ?

Activation function play an integral role in neural network by introducing nonlinearity. This nonlinearity allows neural network to develop complex representation and functions based on the inputs that would not be possible with simple linear regression model.

Types of Activation Functions.

  • Linear Activation Function.
  • Non-Linear Activation Function.
  • Sigmoid or Logistic Activation Function.
  • Tanh or Hyperbolic Tangent Activation Function.
  • Relu (Rectified linear Unit) Activation Function.
  • Leaky ReLU Activation Function.
  • Parameterised ReLU.
  • Exponential Linear Unit
  • Threshoild or Binary Step Function.
  • Softmax Activation Function.
  • Swish Activation Function.

Linear Activation Function.

The Linear activation function, also known as "no activation" or "identity function" (multiple * 1.0), is where the activation is proportional to the input. The function doesn't do anything to the weighted sum of the input, it simply spits out the value it was given.

Non-Linear Activation Function.

The Non-Linear activation function are most used activation functions. Non-linearity helps to makes the graph look something like this. It makes it easy for the model to generalize or adapt with variety of data and to differentoate between the output.

Sigmoid or Logistic Activation Function.

The Sigmoid activation function is also called the logistic function. it is the same function used in the logistic regression classification algorithm. The functo takes any real value as input and outputs the value in the range of 0 to 1 (Larger input is more +ve and smaller input is more -ve). also, as the sigmoid is a non-linear function, the output of this unit would be a non-linear function of the weighted sum of inputs.

Hyperbolic Tangent (TanH)Activation Function.

The hyperbolic tangent activation function is also reffered to simply as the Tanh (also "tanh" & "TanH") function. it is very similar to the Sigmoid activation function and even has the same S- shape. The function takes any real value as input and output value in the range -1 to 1. The biggest advantage of the function is that it produces a zero-centered output.

ReLU (Rectified Linear Unit Activation function)

ReLu Stands for Rectified linear Unit. The main advantage of using the ReLU function over other activation functions is that it does not activate all the neurons at the same time. The ReLU AF for short is a piecewise linear function that will output the input directly if it is positive, otherwise it will output zero.

Leaky ReLU Activation function

Leaky rectified Linear Unit, or Leaky reLU, is a type of activation function based on a ReLU but it has a small slope for negative values insted of a flat slope. The slope coefficient is determined before training, i.e it is not learnt during training. This type of activation function is popular intasks where we may suffer from sparse gradients, for example training generative adversarial networks. Leaky ReLU activate the deactivated neurons.

Parameterized ReLu Activation Function.

A parameteric linear Unit, or PReLU, is an activation function that generalizes the tradictional rectified unit with a slope for negative values. Parameteric relu function used when leaky relu function still fails at solving the problems of dead neurons.

Exponential Linear Unit

This is an another type of relu that modifies the slope of -ve part of the function. It provides the dead relu problem. It increase the computational timing exponential problem.

Threshold or Binary Step Activation Function

A binary step function is a threshold based activation function. If input value is above or below certain threshold, the neuron is activated and send exactly the same signal to the next layer. The activation function compares the input value to the threshold value.

Softmax Activation Function

Softmax is used for multiclassification problems where class membership is required more than 2 class labels. Softmax function is used in output layer of neural network that predict multinominal probablity distribution. It is an Non-linear activation function. It is used in ANN and CNN, Softmax is a combination of sigmoid.

Swish Activation function.

The swish function, or the self gated function, is just another activation function proposed by Google. Swish activation is used in the Long Short-Term Memory (LTST) neural networks, which are used extensively in sequence prediction and likehood problems.
