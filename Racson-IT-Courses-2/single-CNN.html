What is Convolutional Neural Network (CNN) ?

CNN is a subset of Machine Learning and it is a type of Artificial neural network. A CNN is a network architecture for deep learning that learns from data. CNN are particularly useful for finding patterns images to recognize objects, classes and categories. They can also be quite effective for classifying audio, time-series and signal data.

CNN is powerful tool but requires millions of labelled data points for training. A Convolutional Neural Network that has one or more convolutional layers and is used mainly for image processing, classification, segmentation and also for other autocorrelated data. A convolutionan is essentially sliding a filter over the input.

Architecture of Convolutional Neural Network

The Architecture of CNN is basically a list of layers that transforms the 3-dimensional i.e width, height and depth of image volume into 3-dimensional output volume. One important point to noe here is that, every neuron in the current layer is connected to a small patch of the output from the previous layer which is like overlaying a N * N filer on the input image.

It uses M filters, which are basically feature extractors that extract features like edges, corner and so on.
Following are the layers [INPUT - CONV - RELU - POOL - FC ] That are use to construct Convolutional neural network (CNNs)

  1. Input Layer

    Input Layer in CNN should contain image data. Image data is representated by three dimensional matrix as we saw earlier. You need to reshape it into a single column. Suppose yo have image of dimension 28 * 28 = 784, you need to conver it into 784 * 1 before feeding into input.

  2. Convolutional Layer

    The Convolutional layer is the core building block of a CNN. It contains a set of filters (or kernels), parameters of which are to be learned throughout the training. The size of the filters is usually smaller than the actual image. Each filter convolves with the image and creates an activation map.

  3. RELU Layer

    Relu also called rectified linear unit layer, that applies an activation function to the output of previous layer. In other manner, a non-linaerity would be added to the network by RELU.

  4. pooling Layer

    The purpose of the pooing layer is to reduce the dimensions of the hiddenlayer by combining the outputs of neurons clusters at the previous layer into a single neuron the next layer.

  5. Fully Connected Layer

    Fully connected layer is simoly, feed forward neural networks. Fully conneced layers from the last few layers in the network. The input to the fully connected layer is the output from the final pooling or Convolutional Layer, which is flattened and then fed into the fully connected layer.

Hyperparameter of Convolutional Neural Network.

  • Filter

    A filter acts as a single templet or pattern, which, when convolved across the inputs, finds similarities between the stored template & different locations/regions in the input image.

  • Kernel

    In the Convolutional Neural Network, the kernal is nothing but a filter that is used to extract the features from the images. The kernal is a matrix that moves pver the input data, performas the sub=region of input data, and gets the output as the matrix of dot products.

  • Stride

    Stride denotes how many steps we are moving in each steps in convolutional. By default it is one Convolution with Stride 1. When the Stridee is 1 then we move the filers to 1 pixel at a time. When the stride is 2 then we move the filters to 2 pixels at a time and so on.

  • Padding

    Padding is a term relevant to convolutional neural networks as it refers to the amount of pixles added to an image when it is being processed by the Kernel of a CNN. For example if the padding in a CNN is set to Zero, then every pixel value that is added will be of value zero.

  • Pool Size

    The size of the pooling operation or filter is smaller than the size of the feature map specially it is almost always 2 * 2 pixles applied with a stride of 2 pixels.

  • Flattering

    Flattering is used to convert all the resultant 2_Dimensional arrays from pooled feature maps into a single log continuous linear vector. The flattened matrix is fed as input to the fully connected layers to classify the image.

Why we use a Convolutional Neural Network ?

There are multiple benifits of using thismodel as the state of art neural network. As it can be used in various fields and platform major tasks like facial recognition, analyzing documents, understanding climate and image recognitio and object detectio etc. DeepLearning has helped enormously in advancement of the science fields and CNN is the most popular one as it attains the benifits of providing maximum performance and efficiency.

Applications of Convolutional Neural Network

  1. Health Care

    CNN can examine thousands of visual reports to detect any anomalous condition in patients, such as the presense of cancer cells.

  2. Automotive

    CNN Technology is powering research into autonomous vehicles and self-driving cars.

  3. Social Media

    Social media platforms use CNNs to identify people in the User's photograph and help to tag their friends.

  4. Retail

    E-commerce platforms that incorporate visual search allow brands to recommend items that are likely to appeal to a shopper.

  5. Facial recognition for law enforcement

    Generative adversial network (GANs) are used to produce new images that can then be used to train deep-learning models for facial recognition.

  6. Audio Processing for Virtual assistants.

    CNN in virtual assistants learn and detect users-spoken keywords and process that input to guide their actions and respond to the user.

Advantages of Convolutional Neural Network.

The main advantage of CNN compared to its predecessors is that it automatically detects the important features witout any human supervision. For example, given many pictures of cats and dogs it learns distinctive features for each class by itself. CNN is also computationally efficient.

Disadvantage of Convolutional Neural Network

  • A lot of Training data is needed.
  • CNNs tends to much slower.
  • Training process takes a long time.
  • Faild to encode the position and orintation of objects.
